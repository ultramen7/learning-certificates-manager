== Description ==

Manage and display learning certificates grouped by institution. 
Includes:
* Custom post type "Certificates"
* Grouping via "Certificate Groups" taxonomy
* Media Library file picker per certificate
* Modern card layout with hover tooltips and image preview.

== Usage ==

1. Go to "Certificates" > "Certificate Groups" and add your groups (e.g., School certificate, University A, University B).
2. Go to "Certificates" > "Add New" to create certificates:
   * Title = certificate name
   * Content = description (shown in tooltip)
   * Certificate File = select a file from Media Library
   * Assign a Certificate Group.
3. Add the shortcode [learning_certificates] to any page.
